Sentence Structure

Filipino sentence structure is generally

Verb-Subject-Object (VSO)

This is the "normal" or natural way of speaking.

English follows the Subject-Verb-Object (SVO) but it Filipino this structure tends to sound more formal, literary, or emphatic.

VSO

tumatakbo ang lalaki / the man is running

tumatakbo ang batang lalaki / the boy is running

tumatakbo ang mga lalaki / the men are running

tumatakbo ang mga batang lalaki / the boys are running

tumatakbo ang babae / the woman is running

tumatakbo ang batang babae / the girl is running

tumatakbo ang mga babae / the women are running

tumatakbo ang mga babae lalaki / the girls are running
