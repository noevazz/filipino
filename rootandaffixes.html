Root And Affixes

In Filipino, roots (ugat) and affixes (panlapi) are essential parts of word formation.

A root word is the base form of a word that carries the core meaning. It can stand alone or be modified with affixes to create new words.

An affix is a word part (prefix, infix, or suffix) that is added to a root word to modify its meaning.

Prefix (mag-)

Added before the root word.

Example:

magluto / to cook

In this example mag- was added at the beginning of the root word luto (cook).

Infix (um)

Added inside the root word.

Root word in this example: kain / eat

kumain / ate

In the example above um was inserted into kain

More about the um infix

The -um- infix helps to indicate the tense of the verb:

Root word in this example: takbo / run

  • Past Tense: tumakbo (we already saw thi in the kumain example)
  • Present Tense: tumatakbo

    Here "-um-" was added and the first syllable was repeated

  • This is not related to the -um- infix but just for your knowledge. The future tense is created repeating the first syllable:

    tatakbo

If you struggled to understand the present tense example, here is a further explanation:

  • root word: takbo
  • add um: tumakbo (past tense)
  • now repeat the first syllable of the root word: tumatakbo (present tense)

Suffix (-an)

Added at the end of the root word.

Example:

The suffix -an is often used to indicate a place or location related to the root word. It typically turns the root into a noun that refers to a place where the action of the root word occurs.

lakaran / a place for walking

In this example -an was added at the end of the root word lakad (walk).

Another example:

Kainan / a place to eat

In this example -an was added at the end of the root word kain (eat).

Circumfix (pag-ROOT-an)

Combination of prefix and suffix.

Example:

pag-aralan / to study something

In this example pag- and -an surround the root aral (study).

Prefix (Ma-)

The prefix ma- is often used to indicate a state or condition, capability, or emotion related to the root word.

Examples:

masaya / happy

In this example, ma- was added to the root word saya (happiness) to describe the state of being happy.

mahirap / difficult

In this example, ma- was added to the root word hirap (difficulty) to describe the state of something being difficult.

Prefix (Nag-)

The prefix nag- is often used to indicate an ongoing or completed action in the past, typically in reference to a specific person or thing.

Example:

nagluto / cooked (already)

In this example, nag- was added to the root word luto (cook), indicating that the action was completed in the past.

Reduplication

Reduplication is the process of repeating part or all of the root word. It can modify the meaning, often indicating plurality, intensity, or frequency.

Examples:

takbo-takbo / running around

In this example, the root word takbo (run) is repeated to indicate running around or frequently running.

saya-saya / very happy

In this example, the root word saya (happy) is repeated to indicate being very happy or extremely happy.
